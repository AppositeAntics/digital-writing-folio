# digital-writing-folio

**About me!**

My favourite thing to write is fiction. I also write lots of non fiction, but usually for more practical reasons, like giving speeches or contributing to a debate.

What are my tools as a writer? Well, I use lots of different tools. If I use any one for too long, I start to stagnate. I like to use google docs, and word, sometimes notepad and the notes app, and often I will just write things out physically. 

I think this profusion of tools is extraordinarily useful. Words are everywhere, used for all sorts of things. And if words are everywhere, then writing should be everywhere too. 

I’m excited for digital writing because I like games studies and learning about digital media. I want to learn how to make some of my own.
